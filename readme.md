Description
===========
Adds a shortcode {markdown} for generating markdown to HTML with the help of parsedown. (see https://github.com/erusev/parsedown)
It can handle html coded content (e.g. from MCE or JCE Editor) and converts it to expected ascii text before generating the page.
For what is markdown see https://de.wikipedia.org/wiki/Markdown
For syntax of markdown see e.g. https://www.markdownguide.org/basic-syntax/
